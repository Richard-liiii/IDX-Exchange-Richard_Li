# California Property Close Price Prediction

## Project Overview

This project develops machine learning models to predict residential property close prices using California MLS/CRMLS housing data. The workflow focuses on data preprocessing, exploratory data analysis, feature engineering, regression modeling, model evaluation, and prediction of property sale prices.

## Main Tasks

* Cleaned and preprocessed California residential property data
* Performed exploratory data analysis (EDA) to understand housing market characteristics
* Engineered predictive features from property attributes
* Built and compared multiple regression models for property price prediction
* Evaluated model performance using metrics such as R², MAPE, and MdAPE
* Documented the complete workflow for reproducibility and future development

## Repository Structure

* **README.md:** Project overview and reproduction instructions
* **src/:** Python source code
* **notebooks/:** Jupyter notebooks for data analysis and modeling
* **data/:** Sample or non-confidential datasets
* **results/:** Model evaluation results and visualizations

## How to Reproduce

1. Download or clone this repository.
2. Install the required Python packages.
3. Open the notebook in the **notebooks/** folder.
4. Run the data preprocessing, feature engineering, model training, and evaluation steps.
5. Review model outputs and visualizations in the **results/** folder.
